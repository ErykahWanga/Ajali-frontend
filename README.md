# Incident Reporting System

A web application for reporting and managing incidents with geolocation capabilities.

## Features

- Create, read, update, and delete incidents
- Geolocation mapping with Google Maps integration
- Media uploads with preview functionality
- Filter and sort incidents by status and category
- Responsive design for all device sizes

## Installation

1. Clone the repository
2. Install dependencies: `npm install`
3. Create a `.env` file based on `.env.example`
4. Start the development server: `npm start`

## Environment Variables

- `REACT_APP_API_BASE_URL`: Base URL for the API
- `REACT_APP_GOOGLE_MAPS_API_KEY`: Google Maps API key

## Available Scripts

- `npm start`: Runs the app in development mode
- `npm test`: Launches the test runner
- `npm run build`: Builds the app for production
